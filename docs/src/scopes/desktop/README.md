# Desktop Environments

Wayland compositor scopes — each self-activates via a hostSpec flag.

| Scope | Flag | Compositor |
|-------|------|-----------|
| [Niri](niri.md) | `useNiri` | Scrollable-tiling (NixOS-only) |
| [Hyprland](hyprland.md) | `useHyprland` | Dynamic tiling |
| [GNOME](gnome.md) | `useGnome` | Full desktop environment |
